# Surebet LOZZY — versão celular/PWA

## Android
1. Hospede esta pasta em um link HTTPS (GitHub Pages, Netlify, Vercel etc.).
2. Abra o link no Chrome ou Edge do celular.
3. Toque em "Instalar app" ou use o menu ⋮ → "Instalar app" / "Adicionar à tela inicial".
4. Envie os arquivos HAR/ZIP para o celular e selecione dentro do app.

## iPhone
1. Abra o link no Safari.
2. Toque em Compartilhar.
3. Toque em "Adicionar à Tela de Início".

## Observações
- Abrir como arquivo local ainda funciona em alguns celulares, mas o modo instalável PWA funciona melhor quando hospedado em HTTPS.
- HAR muito grande pode travar celulares mais fracos.
- Capturar HAR ainda é mais fácil no computador.
